Alanta.Mvc : View composition for ASP.NET MVC 1.0
-------------------------------------------------

This project provides support for view composition from a controller in ASP.NET MVC 1.0 websites. It allows you to load any ASP.NET user control (.ascx) or custom control into any of the ContentPlaceHolders of a master page.

Main features:
- Compose views using master pages and controls (no .aspx required)
- Load views by type (custom controls) or path (user controls)
- No strict requirements on site structure
- Strong-typed routing for controllers
- Use MVC with plain ASP.NET 3.5 websites
- No view page required (.aspx)

Please check http://blog.alanta.nl for more information.

Requirements
------------

- Microsoft .NET 3.5 SP1
- Microsoft ASP.NET MVC 1.0
- A development environment for .NET (Visual Studio 2008)

Deployment
----------

Use XCopy deployment; no installation required.
